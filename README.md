# Healthcare Operations & Patient Analytics using SQL and Power BI

## Project Overview
This project analyzes hospital data to understand patient demographics, appointments, diagnoses, billing, and insurance coverage. SQL was used to query and analyze the data, and Power BI was used to create interactive dashboards to support operational and financial decision-making.

## Objectives
- Analyze appointment completion and no-show rates  
- Evaluate doctor performance and revenue contribution  
- Examine billing and insurance coverage patterns  
- Identify common diagnoses and patient trends  

## Tools Used
- SQL (MySQL / PostgreSQL)  
- Power BI  

## Key Deliverables
- Structured healthcare database  
- SQL analysis queries  
- Power BI dashboard  
- Project documentation
